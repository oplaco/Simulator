# Simulator

## Description
This project is an open-source flight simulator designed to integrate both aeronautical and software engineering principles. Utilizing NASA's WorldWind SDK and Java, it offers a flexible, modular platform aimed at fostering collaboration and continuous development, the simulator allows researchers and developers to easily extend its capabilities and experiment with new algorithms and models. Ideal for educational purposes, this project combines advanced simulation techniques with an intuitive graphical interface.

## Instalation
Ensure Java 11 is installed.
You can check if Java 11 is installed by running: -java version
There are no additional dependencies to install as they are already committed in the repository.
Clone the repository and you are good to go.

## Run a demo
To run the project, open your preferred IDE (e.g., NetBeans, IntelliJ IDEA, Eclipse), navigate to the GUI.java file in the project directory, and run it.
